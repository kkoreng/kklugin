# kklugin

Gradle plugin for Minecraft plugin development. Supports Bukkit/Spigot and Paper servers.

## Plugins

| Plugin ID | Description |
|-----------|-------------|
| `com.kkoreng.kklugin.bukkit` | Bukkit/Spigot plugin development |
| `com.kkoreng.kklugin.paper` | Paper plugin development |
| `com.kkoreng.kklugin.run` | Run Minecraft server during development |

## Usage

### Bukkit / Spigot

```kotlin
// settings.gradle.kts
pluginManagement {
    repositories {
        gradlePluginPortal()
    }
}
```

```kotlin
// build.gradle.kts
plugins {
    id("com.kkoreng.kklugin.bukkit") version "0.0.1"
}

kklugin {
    plugin {
        name = "MyPlugin"
        version = "1.0.0"
        main = "com.example.MyPlugin"
        minecraftVersion = "1.21"
        description = "My awesome plugin"
        authors = listOf("yourname")
    }
}
```

### Paper

```kotlin
plugins {
    id("com.kkoreng.kklugin.paper") version "0.0.1"
}

kklugin {
    plugin {
        name = "MyPlugin"
        version = "1.0.0"
        main = "com.example.MyPlugin"
        minecraftVersion = "1.21"
    }
}
```

### Run Server

```kotlin
plugins {
    id("com.kkoreng.kklugin.run") version "0.0.1"
}

kklugin {
    runServer {
        minecraftVersion = "1.21"
        platform = ServerPlatform.PAPER
        port = 25565
        acceptEula = true
    }
}
```

#### Run with Proxy

```kotlin
kklugin {
    runProxy {
        minecraftVersion = "1.21"
        platform = ProxyPlatform.VELOCITY
        port = 25577

        backends("lobby") {
            minecraftVersion = "1.21"
            platform = ServerPlatform.PAPER
            port = 25565
            acceptEula = true
        }
    }
}
```
